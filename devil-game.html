The Devil's Table

悪魔の遊戯室

『悪魔と66の契約』——大アルカナの番号を足して、66に近づけ。欲張った札には、私の印が残る。

あなたの勝利 0
悪魔の勝利 0
契約中 0
悪魔の機嫌 平静

遊戯の規則

悪魔の館では、66に近づいた者ほど奥の間へ招かれる。
そこには、まだ名のない祝福が眠っているという。ふむ、君の名で呼ばれる日が来るかもしれんな。

大アルカナのカード番号を足していきたまえ。
合計を66以下に保ちながら、できるだけ66へ近づけるのだ。

最初に、君には二枚のカードが配られる。
私にも二枚のカードが配られるが、一枚は表向き、一枚は伏せ札だ。

君の番では、「カードを引く」か「ここで留まる」を選べる。

「カードを引く」を選べば、新しいカードを一枚引き、合計に加える。
もう一歩、奥へ進むということだ。

「ここで留まる」を選べば、それ以上カードを引かず、その合計で勝負する。
よかろう、慎みもまた美徳ではある。

君が留まれば、次は私の番だ。
私の番のはじめに伏せ札を開き、必要ならさらに札を引く。

66以下で、より66に近いほうが勝者となる。
同点の場合は、悪魔が勝利する。そういう規則なのだよ。

66を越えた時、悪魔は契約を差し出すことがある。
契約すると、66を越えた分だけ66から手前へ戻される。契約後はその数で留まり、最後に引いた札には悪魔の印が残る。

契約の確認

悪魔の印を受けたカードを確認できます。契約を解くには、あなたの勝利をひとつ支払う必要があります。支払った勝利は、悪魔の勝利として数えられます。

記録の初期化

記録を白紙に戻すかね？

君の勝利、私の勝利、契約の印はすべて消える。
帳簿は閉じられ、札は以前の姿へ戻る。

だが、白紙とは無垢のことではない。
積み上げたものを消した、という事実だけは、君の中に残るだろう。

この操作は取り消せない。
本当に、遊戯室の記録を初期化するかね？

契約の提示

66を越えたな。
契約するなら、越えた分だけ手前へ跳ね返してやろう。
代わりに、最後に引いた札には私の印が残る。
