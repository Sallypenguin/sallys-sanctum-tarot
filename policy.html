サイトポリシー

Sally’s Sanctum Tarot は、個人制作の静的タロットWebアプリです。 本ページでは、著作権、制作過程の透明性、保存データ、外部サービス、免責事項について説明します。

最終更新日：2026年5月6日

このサイトについて

本サイトは、ステンドグラス風の大アルカナ22枚を使い、カードの象徴を通じて利用者自身の状態や問いを見つめるために作られた個人制作サイトです。 未来の出来事を断定することや、他者の意思を確定的に読み取ることを目的としていません。

SNSで広く拡散するためのサービスではなく、個人的な内省、または信頼できる範囲での静かな参照を想定しています。

著作権・利用範囲

本サイトに掲載しているカード画像、文章、カード解釈、画面デザイン、構成等は、制作者が企画・選定・編集・修正し、Sally’s Sanctum Tarot として構成したものです。 これらの素材・構成物に関する権利は、法令上保護される範囲において制作者に帰属します。

本サイトの画像、文章、カード解釈、画面デザイン等の無断転載、再配布、商用利用、改変利用はご遠慮ください。 個人的な閲覧、内省、少人数での参照を想定しています。

生成AIを利用して制作した部分については、利用したサービスの規約および適用される法令の範囲内で扱います。 ただし、AI生成物の著作物性や第三者の権利との関係は、個別の事情によって判断され得るものです。 本サイトでは、権利関係について過度に断定せず、制作者による企画、選定、編集、修正、構成を含む制作物として丁寧に管理します。

制作の透明性

本サイトの制作過程では、カード画像の生成、文章整理、実装補助等に生成AIを補助的に使用しています。 最終的な選定、編集、構成、修正、公開判断は制作者が行っています。

生成AIの出力をそのまま無条件に採用するのではなく、デッキ全体の思想、カードごとの象徴、画面上での見え方、文章の調子、利用者への伝わり方を確認しながら調整しています。

プライバシーについて

本サイトは、アカウント登録、問い合わせフォーム、コメント欄、SNS共有機能、アクセス解析ツール、広告配信機能を設けていません。 制作者が、リーディング内容や閲覧履歴をサーバー上で収集・閲覧する仕組みはありません。

リーディング履歴の保存について

本サイトの「記録」機能は、利用者が明示的に「この読みを記録する」ボタンを押した場合にのみ、リーディング履歴を保存します。 自動的にすべてのリーディングを保存することはありません。

保存される内容には、保存日時、スプレッドの種類、相談テーマ、引いたカード、正位置・逆位置、カードの位置、表示された読み、シナジー情報等が含まれます。 これらの記録は、利用者の端末のブラウザ内に保存されます。本サイトのサーバーや制作者に送信されるものではありません。

記録は、同じ端末・同じブラウザ・同じ閲覧環境で参照できます。 別の端末や別のブラウザには自動で同期されません。 また、ブラウザのデータ削除、サイトデータの削除、プライベートブラウジングの終了、端末やブラウザの設定変更等により、記録が消えることがあります。

履歴ページでは、個別の記録を削除すること、およびすべての記録を削除することができます。 保存された記録を確実に残しておきたい場合は、利用者自身の責任で別途控えを取ってください。

共有端末での利用について

リーディング履歴は、端末内のブラウザに保存されます。 そのため、家族共用端末、職場端末、学校端末など、複数人が同じブラウザ環境を使う場合、他の利用者が履歴を閲覧できる可能性があります。 共有端末で利用する場合は、履歴を保存しないか、利用後に履歴を削除することをおすすめします。

Cookie・トラッキングについて

本サイト自体は、広告配信、行動追跡、アクセス解析を目的としたCookieを設定していません。 また、利用者の行動を分析するためのトラッキングツールは設置していません。

ただし、ブラウザ内の記録機能には Cookie ではなく localStorage を使用しています。 localStorage は、利用者のブラウザ内にデータを保存するためのWeb標準機能です。 ブラウザや端末の設定により、保存、削除、利用可否の扱いが異なる場合があります。

外部サービスについて

本サイトは GitHub Pages によって公開されています。 GitHub Pages の仕組み上、サイトへのアクセス時に、GitHub がセキュリティ目的等のためにIPアドレス等の技術情報を記録する場合があります。 詳しくは GitHub Pages の説明および GitHub Privacy Statement をご確認ください。

また、本サイトでは表示用フォントの一部に Google Fonts を使用しています。 そのため、ページ表示時に訪問者のブラウザから Google Fonts のサーバーへリクエストが送信され、IPアドレス、リクエストURL、ブラウザ情報等の技術情報が Google に送信される場合があります。 Google Fonts のデータ処理については Google Fonts のプライバシーに関する説明 をご確認ください。
